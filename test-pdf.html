Topic 1: Introduction to Multimodal Learning

Multimodal machine learning is the study of computer algorithms that learn from multiple modalities of data. These modalities can include text, images, audio, video, and sensor data. The goal is to build models that can understand and reason across these different types of data simultaneously.

Key challenges in multimodal learning include:

  • Representation: How to represent data from different modalities in a shared space
  • Alignment: How to align information across modalities
  • Fusion: How to combine information from multiple sources effectively
  • Translation: How to translate between modalities
import torch
from torch import nn

class MultimodalEncoder(nn.Module):
    def __init__(self, text_dim, image_dim, output_dim):
        super().__init__()
        self.text_encoder = nn.Linear(text_dim, 256)
        self.image_encoder = nn.Linear(image_dim, 256)
        self.fusion = nn.Linear(512, output_dim)
    
    def forward(self, text, image):
        t = self.text_encoder(text)
        i = self.image_encoder(image)
        fused = torch.cat([t, i], dim=-1)
        return self.fusion(fused)

Note: Early fusion strategies combine raw features before processing, while late fusion strategies process each modality independently before combining.

Topic 3: Modality Alignment Strategies

Alignment refers to the process of mapping data from different modalities into a common representation space. This is crucial for tasks like cross-modal retrieval and multimodal fusion.

Common alignment approaches include:

  • Linear Projection: Simple matrix multiplication to project features
  • Attention Mechanisms: Learn which parts of each modality to attend to
  • Contrastive Learning: Pull matching samples closer, push non-matching apart
  • CCA (Canonical Correlation Analysis): Statistical method for finding correlations
def contrastive_loss(anchor, positive, negative, margin=1.0):
    d_pos = torch.dist(anchor, positive, p=2)
    d_neg = torch.dist(anchor, negative, p=2)
    loss = torch.relu(d_pos - d_neg + margin)
    return loss.mean()

Output: The model learns to minimize distance between aligned pairs and maximize distance between non-aligned pairs.

class EarlyFusion(nn.Module):
    def __init__(self, input_dims, hidden_dim):
        super().__init__()
        total_dim = sum(input_dims)
        self.fc = nn.Sequential(
            nn.Linear(total_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )
    
    def forward(self, *modality_inputs):
        concatenated = torch.cat(modality_inputs, dim=-1)
        return self.fc(concatenated)

Cross-modal retrieval involves searching for data in one modality using a query from another modality. For example, searching for images using text descriptions.

Key applications include:

  • Image-to-Text Captioning
  • Text-to-Image Search
  • Audio-Visual Scene Understanding
  • Multimodal Recommendation Systems

The architecture typically involves:

  1. Separate encoders for each modality
  2. A shared embedding space
  3. Similarity scoring mechanism
  4. Ranking and retrieval pipeline
